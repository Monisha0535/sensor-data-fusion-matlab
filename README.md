# Sensor Data Fusion (MATLAB)

##  Description
This project implements sensor data fusion techniques using MATLAB to improve the accuracy of sensor measurements.

##  Technologies
- MATLAB
- Kalman Filter 

##  Concept
Sensor fusion combines data from multiple sensors (e.g., accelerometer, gyroscope) using algorithms such as Kalman filtering to improve accuracy and reliability.

##  How to Run
1. Navigate to the MATLAB folder
2. Run `moni.m` (main script)
