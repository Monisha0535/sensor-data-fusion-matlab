# Sensor Data Fusion (MATLAB)

##  Description
This project implements sensor data fusion techniques using MATLAB to improve the accuracy of sensor measurements.

##  Technologies
- MATLAB
- Kalman Filter (if used)

##  Concept
Sensor fusion combines multiple sensor inputs (e.g., accelerometer, gyroscope) to produce more reliable results.
